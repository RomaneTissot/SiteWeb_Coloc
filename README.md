# SiteWeb_Coloc

Projet de développement web réalisé dans le cadre de l’électif **Développement Web**.
Ce site présente une colocation située à Marseille à travers différents articles, anecdotes et moments de vie entre colocataires.

---

## Aperçu du projet

Le site a été conçu comme un espace immersif et humoristique permettant de découvrir :

* les colocataires,
* la vie quotidienne dans la colocation,
* différentes péripéties et souvenirs,
* des pages de type “article” racontant certains événements.

Le projet a été entièrement développé avec **HTML**, **CSS** et **JavaScript** sous **VSCode**.

---

## Technologies utilisées

* HTML5
* CSS3
* JavaScript
* MongoDB (ancienne version du livre d’or)

---

## Fonctionnalités

* Navigation entre plusieurs pages/articles
* Mise en page personnalisée
* Intégration d’images et de médias
* Design responsive
* Livre d’or permettant de laisser des avis/commentaires
* Backend destiné à stocker les commentaires utilisateurs

---

## État du livre d’or

Le système de sauvegarde des commentaires n’est actuellement plus fonctionnel.

Le backend utilisait une base de données MongoDB hébergée via une offre gratuite qui a expiré.
L’interface du livre d’or reste néanmoins présente dans le projet.

## Lancer le projet

Le projet est conçu pour être lancé localement avec **Live Server** dans VSCode.

### Étapes

1. Ouvrir le projet dans VSCode
2. Installer l’extension **Live Server**
3. Ouvrir `index.html`
4. Faire :

   ```text
   Open with Live Server
   ```

Le site s’ouvrira automatiquement dans le navigateur.
